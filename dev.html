System Data Inspector
Find out which mysterious System Data on your Mac is taking up so much space.

Python Bash

The System Data Inspector is a a command line application designed for MacOS users to efficiently gauge the space occupied by System Data on their Mac. It provides a total size for predefined system directories and also features a color-coded display for enhanced readability. Users have the option to delve deeper into specific directories and even open them in Finder.
