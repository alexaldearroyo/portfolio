Priorities
Keep track of your most important tasks at a glance!

Python PyQt5

"Priorities" is a task management application for macOS with a color-changing interface based on task count: blue for few tasks, yellow for some, and red for many. It features a user-friendly interface supporting both mouse and keyboard navigation, along with a text-editing function for easy task modifications.

Markdown Viewer
View your markdown text files carefully formatted with this lightweight and compact app.

Python PyQt5

"Markdown Viewer" is an intuitive application designed for MacOS, allowing users to view markdown (.md) and text (.txt) files. Built with Python and PyQt5, this tool offers a sleek and lightweight interface with standout features such as light and dark mode support, tabbed navigation, keyboard shortcuts, and custom themes. It is powered by Mistune for accurate and swift markdown content interpretation.
